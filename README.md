# My Vim Configuration

A professional, minimal and dark-themed Vim setup tailored for web development.

## Features

- File explorer (NERDTree)
- VSCode-style auto-completion (coc.nvim)
- Auto-closing tags & brackets
- Linting with ALE
- Emmet support
- Gruvbox dark theme
- Git integration
- Fuzzy file search with fzf
